# stock
**Implemented API Rest for rapidly built warehouses
In the appendix, it was validated and sirive of data. 
Migrations are made with changes in tables**
##
**Stack:**
- Flask
- Postgres
- Flask-Sqlalchemy
- Flask-Migrate
- Pydantic
- Doker
##
**Routes (GET, POST, PUT, DELETE):**
- */articles/*
- */specifications/*
- */groupSpec/*
- */categories/*
- */articleSpec/*
- */categoryArticle/*
